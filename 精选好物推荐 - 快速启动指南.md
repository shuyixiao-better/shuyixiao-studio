## 📝 快速配置

### 第一步: 替换佣金链接

编辑 `docs/.vitepress/data/affiliate.ts`:

```typescript
{
  id: 'jetbrains-toolbox',
  title: 'JetBrains 全家桶',
  // ...其他配置
  link: 'https://你的佣金链接.com',  // ← 修改这里
}
```

### 第二步: 调整产品数量

修改 `featured: true` 的产品数量 (建议 2-4 个):

```typescript
{
  // ...
  featured: true,   // 首页显示
  active: true,     // 激活状态
  priority: 10,     // 排序优先级 (10最高)
}
```

### 第三步: 自定义说明文字 (可选)

```typescript
export const affiliateConfig = {
  disclosure: {
    long: '你的自定义说明...'
  }
}
```
