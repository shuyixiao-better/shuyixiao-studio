# AdSense 复审前检查清单

## 1. 内容质量

- 确保公开页面中不再出现“正在撰写中”“草稿”“模板示例”等占位内容。
- 至少准备一批完整且可读的高质量文章（建议先保证 20+ 篇）。
- 删除或 noindex 近重复文章，避免同主题多版本重复。

## 2. 合规与信任

- 确认以下页面可直接访问：
  - [隐私政策](/policies/privacy-policy)
  - [服务条款](/policies/terms)
  - [免责声明](/policies/disclaimer)
  - [联系我们](/policies/contact)
- 联系方式可用，邮箱可接收外部邮件。

## 3. 抓取与索引

- 在 Search Console 提交最新 sitemap。
- 对核心栏目页和高质量文章页手动请求重新抓取。
- 复查 noindex 页面是否已从索引中逐步退出。

## 4. 广告与体验

- 每页广告密度控制在合理范围，避免首屏广告过重。
- 移动端确认正文可读性优先，不被广告或弹层干扰。
- 页面加载无明显报错，关键内容可稳定渲染。

## 5. 复审节奏

- 完成以上整改后，建议等待 7-14 天再点击“申请审核”。
- 复审期间继续发布高质量原创内容，避免仅做技术性修补。
