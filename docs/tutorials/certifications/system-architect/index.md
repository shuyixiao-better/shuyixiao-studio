# 系统架构设计师（高级）

系统架构设计师是全国计算机技术与软件专业技术资格（水平）考试（简称"软考"）的高级资格考试，主要面向具备系统架构设计能力的资深技术人员。

## 考试基本信息

| 项目 | 内容 |
|------|------|
| 考试级别 | 高级 |
| 考试时间 | 每年5月、11月（具体以官方公告为准） |
| 考试时长 | 综合知识：150分钟<br>案例分析：90分钟<br>论文：120分钟 |
| 合格标准 | 三科均需达到45分（满分75分） |
| 成绩有效期 | 当次考试有效，必须一次性通过三科 |

## 考试科目

### 1. 综合知识（上午）

- **题型**：单项选择题（75题）
- **分值**：75分
- **时长**：150分钟
- **主要考点**：
  - 计算机组成与体系结构
  - 操作系统
  - 数据库系统
  - 计算机网络
  - 系统安全
  - 软件工程
  - 项目管理
  - 法律法规与标准化

### 2. 案例分析（下午）

- **题型**：问答题（通常4-5道大题，选做）
- **分值**：75分
- **时长**：90分钟
- **主要考点**：
  - 系统架构设计
  - 软件架构设计
  - 数据库设计
  - 网络架构设计
  - 嵌入式系统设计

### 3. 论文（下午）

- **题型**：论文写作（2小时，通常二选一）
- **分值**：75分
- **时长**：120分钟
- **字数要求**：2000-2500字
- **常见主题**：
  - 软件架构设计
  - 系统建模
  - 分布式系统设计
  - 大数据架构
  - 微服务架构

## 备考规划

### 时间规划（建议3-4个月）

```
第1-4周：综合知识学习
├── 计算机组成与体系结构
├── 操作系统
├── 数据库系统
└── 计算机网络

第5-8周：综合知识学习（续）
├── 系统安全
├── 软件工程
└── 项目管理

第9-10周：案例分析专项训练
├── 架构设计模式
├── 数据库设计
└── 网络设计

第11-12周：论文写作准备
├── 论文模板准备
├── 常见主题素材积累
└── 真题练习

第13-16周：冲刺复习
├── 历年真题模拟
├── 查漏补缺
└── 考前冲刺
```
