# 登科录

> 登科及第，学无止境。此录专载各类技术认证考试之备考心得与学习笔记。

## 专栏简介

「登科录」取自古时科举登科之意，记录技术人在专业认证道路上的攀登历程。无论是软考高级、云服务认证，还是各类专业技能证书，这里都将沉淀备考经验、学习笔记与心得感悟。

## 正在备考

### 系统架构设计师（高级）

全国计算机技术与软件专业技术资格（水平）考试中的高级资格认证，是系统架构领域的权威认证。

- **考试科目**：综合知识、案例分析、论文写作
- **考试时间**：每年5月、11月
- **合格标准**：三科均需达到45分（满分75分）

[开始学习 →](/tutorials/certifications/system-architect/)

## 备考心法

::: tip 备考建议
1. **制定计划**：根据考试大纲，合理分配学习时间
2. **理解为主**：不要死记硬背，要深入理解原理
3. **多做真题**：历年真题是最好的练习材料
4. **论文准备**：提前准备论文素材和模板
:::
