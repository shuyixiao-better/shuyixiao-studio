## 💭 使用建议

::: tip 设计素材使用原则
1. **注意版权** - 商业使用前确认授权协议
2. **保持风格统一** - 同一项目使用统一风格的素材
3. **适度使用** - 避免过度设计，保持简洁
4. **优化性能** - 压缩图片，使用合适的格式
5. **建立素材库** - 收藏常用素材，提高效率
:::

::: warning 版权注意事项
- ✅ **CC0协议**: 完全免费，商业使用无限制
- ⚠️ **CC BY**: 需要署名作者
- ⚠️ **个人使用**: 仅限个人项目
- ❌ **商用需授权**: 商业使用需购买授权
:::
