## 💭 学习建议

::: tip 如何高效学习视频课程
1. **选择合适的课程** - 根据自己的水平和目标选择
2. **制定学习计划** - 每天固定时间学习，保持节奏
3. **动手实践** - 边看边敲代码，不要只看不练
4. **做好笔记** - 记录重点和疑问
5. **项目实战** - 学完知识点要应用到项目中
6. **复习巩固** - 定期回顾，温故知新
:::

::: info 视频课程学习策略
- 🎯 **入门课程** - 1.5倍速观看，快速过一遍
- 📚 **进阶课程** - 正常速度，边看边实践
- 🔬 **深度课程** - 0.75倍速，反复观看
- 💡 **遇到难点** - 暂停思考，查资料，做笔记
- 🔄 **重要内容** - 反复观看，直到理解
:::

├─ Java基础 (B站免费课程)

├─ 数据库与缓存 (极客时间)
└─ 微服务架构 (极客时间)

全栈开发路径
├─ CS50计算机基础 (哈佛公开课)

├─ 后端开发 (Udemy实战)

算法与面试
├─ 算法基础 (普林斯顿Coursera)
├─ LeetCode刷题 (B站代码随想录)
