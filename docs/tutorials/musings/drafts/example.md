# 示例：写作草稿存放

> 这是一篇示例文章，展示如何使用草稿功能

## 草稿用途

"随笔草稿"目录适合存放：

### ✍️ 未完成的文章
- 还在撰写中的技术文章
- 需要补充内容的草稿
- 待完善的教程大纲

### 📝 临时记录
- 会议记录和要点整理
- 项目规划和想法梳理
- 待办事项和计划清单

### 🎯 内容大纲
- 系列文章的规划大纲
- 课程/教程的内容框架
- 项目文档的结构设计

## 工作流程建议

1. **快速记录** - 先把想法记下来，不用在意格式
2. **逐步完善** - 有时间时慢慢补充和完善内容
3. **分类整理** - 成熟后可以转移到对应的专栏
4. **定期清理** - 删除过时或不需要的草稿
